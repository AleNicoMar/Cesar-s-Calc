# Cesar-s-Calc

Comandos de acción:

Comando 'cifrar' ó 'c':
	- Escribo mensaje a cifrar.
	- Ingreso clave. *
	- Procesa Cifrado.
	- Escribe archivo.

Comando 'descifrar' ó 'd':
	- Escribo mensaje cifrado.
	- Ingreso clave de cifrado. *
	- Procesa Descifrado.
	- Escribe archivo.

* Las claves deben ser letras del alfabeto español, según su posición en el mismo('a' para 1, 'b' para 2, etc.)
